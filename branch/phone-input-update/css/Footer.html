Footer

The footer, accessible via CSS

If your page has either neptune.css or both neptune/core and neptune/footer, the following HTML structure will render the TransferWise footer.

Ideally you should use a footer which has been packaged for re-use:

  • For React projects, use @transferwise/public-navigation. For styles, ensure the page includes PublicNavigation.css.
  • For Java projects with Thymeleaf templates, use @transferwise/web-components. For styles, ensure the page includes the styles referred to above.
