Neptune Design System – Web

Documentation for building interfaces on web using Neptune components and other resources. Built by TransferWise.

React components v23.0.1

For every Neptune component, we have a page for it with an interactive block where you can edit the code and see a preview of the component live. You can copy this code directly to your project. To learn how to contribute, go here. For more details about usage and bundle distribution please check the detailed page

Install

yarn add @transferwise/components

Use

import { Button } from '@transferwise/components';

CSS v0.13.0

We will have extra documentation on the CSS library soon, in the meantime, here's how you can use it on your project. For more details, here's the source.

Install

yarn add @transferwise/neptune-css

Use

In the dist directory you can find bundles for the individual components, variables, and other common utilities. Use them as individual components or import the whole library.

// Individual
import '@transferwise/neptune-css/dist/css/buttons.css';

// Full bundle
import '@transferwise/neptune-css/dist/css/neptune.css';

Usage guidelines

Guidelines on when to use components are shared across all platforms and documented on a common repository. Please report cross-platform problems and ask questions by creating an issue.
